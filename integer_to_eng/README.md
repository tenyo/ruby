## Coverter from integers to English phrases
Based on http://rubyquiz.com/quiz25.html

Extends the Integer class with a to_eng method

### Demo:
    $ ruby init.rb

    Enter a number: 
    2014
    two thousand and fourteen
    
    Enter a number: 
    -15
    negative fifteen
    
    Enter a number: 
    100000000000000
    one hundred trillion
